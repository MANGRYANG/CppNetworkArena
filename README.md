## 🧭 프로젝트 개요

본 프로젝트는 **C++ 및 Boost.Asio를 활용한 실시간 클라이언트-서버 네트워크 기반 아레나 게임 개발 프로젝트**입니다.

프로젝트 주요 목표는 다음과 같습니다.

- C++를 활용한 게임 클라이언트-서버 구조 설계
- Boost.Asio 기반 비동기 TCP 네트워크 통신 구현
- 클라이언트 입력을 서버로 전송하고, 서버 권위 상태를 기준으로 Player 위치 갱신
- 공격 요청과 서버 Hit 판정을 통해 실시간 게임 시스템의 기본 흐름 구현

## 🔧 사용 기술

- C++ : 서버 / 클라이언트 핵심 로직 구현
- Boost.Asio TCP : 비동기 네트워크 통신, Session 기반 연결 관리
- DirectX 11 : 아레나 디버그 뷰 렌더링
- Dear ImGui : RTT, Packet Log 등 네트워크 상태 시각화
